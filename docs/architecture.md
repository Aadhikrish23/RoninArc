# RoninArc Architecture

## Overview

RoninArc is a desktop-first game library, launcher, and tracking platform.

Tech Stack:

Frontend:

* React
* TypeScript
* Vite
* Chakra UI

Backend:

* Node.js
* Express
* TypeScript
* MongoDB

Desktop:

* Electron

Authentication:

* JWT Authentication
* AuthContext
* Protected Routes
